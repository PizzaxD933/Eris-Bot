🤔 ¿Cómo funciona el bot?

El bot cuenta con múltiples sistemas y comandos. La mejor forma de comenzar es usando .menu para ver las categorías disponibles. También puedes revisar .perfil para consultar tus datos, progreso y estadísticas dentro del sistema.

🎲 ¿Cómo funciona el sistema gacha?

Al usar .rw gastarás ¥3000 por intento. Dependiendo de tu suerte podrás obtener distintos personajes. Algunos son más comunes y otros más difíciles de conseguir. El resultado es completamente aleatorio.

👤 ¿Por qué no puedo reclamar mi personaje?

Porque al utilizar #rw el personaje se reclama automáticamente para tu colección. No necesitas usar un comando adicional para reclamarlo.

🎟️ ¿Para qué sirven los tickets?

Permiten obtener personajes aleatorios relacionados con una serie específica. Puedes buscar series usando .serie y aprovechar los tickets para enfocar tus recompensas.

💰 ¿Por qué no puedo robar?

El sistema no permite robo de dinero entre usuarios. La economía está diseñada para evitar pérdidas por acciones de otros jugadores.

🏦 ¿Por qué no puedo depositar dinero?

No existe un banco funcional dentro del sistema porque nadie puede robarte tu dinero. Por ello tampoco es necesario protegerlo mediante depósitos.

🏧 ¿Cómo retiro dinero del banco?

No hay sistema de banco ni depósitos, por lo tanto tampoco existe un sistema de retiros.

🆓 ¿El bot es gratuito o de paga?

El sistema es gratuito. Los servidores y funciones disponibles para usuarios no requieren pagos para ser utilizados.

🔧 ¿Por qué no funciona mi bot?

Verifica que tu sesión siga vinculada. Si tienes problemas, desvincula la sesión y vuelve a generar un nuevo código de conexión utilizando los comandos correspondientes.
